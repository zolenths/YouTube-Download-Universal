# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-28

### Added
- Initial release of YouTube Download Universal.
- Support for MP3 and FLAC audio formats.
- Retro-Terminal UI with dark/light mode support.
- Safety Gate system to prevent IP bans.
- Built-in proxy and anti-ban configuration.
- Automated setup for yt-dlp and ffmpeg.
- Multi-platform support (Windows, Linux).
